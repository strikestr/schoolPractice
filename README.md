# schoolPractice

### Функционал
- Функция lighten(сolor, percent):
  Делает полученный цвет светлее.
  - Получаемые данные:
    - color: str (#xxxxxx)
    - percent: int
- Функция darken(сolor, percent):
  Делает полученный цвет темнее.
  - Получаемые данные:
    - color: str (#xxxxxx)
    - percent: int
